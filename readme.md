# 🖼 Image Compressor UI

A simple and modern **Image Compression Web App** built using **HTML, CSS, and Vanilla JavaScript**.  
It allows users to upload an image, preview it, compress it using a quality slider, and download the optimized image.

---

## ✨ Features

- 🖼 Upload image and preview instantly
- 🎚 Adjust compression quality (10% - 100%)
- 📉 Compress image in browser (no server)
- 📥 Download compressed image
- 📱 Fully responsive design
- 🎨 Clean modern UI
- ⚡ Fast and lightweight

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Canvas API)
